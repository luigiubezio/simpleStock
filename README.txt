Application can be configured using:
* in/stockPrices.csv: to modify the given sample data
* in/trades.csv: to add a list of trades to be recorded, each trade is loaded in memory using a time frame delay
* in application.properties the following can be set:
** time frame delay in seconds (SEC_LOAD_DELAY)
** ALE ticker price (TEA_PRICE)
** POP ticker price (POP_PRICE)
** ALE ticker price (ALE_PRICE)
** GIN ticker price (GIN_PRICE)
** JOE ticker price (JOE_PRICE)

Launching Starter class, data is evaluated

A test suite is given
